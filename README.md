# GeoCSV QGIS plugin

## Overview

After loading Delimited Text Layer (like CSV files) in QGIS an user cannot edit and interact with it as a vector layer at all. This simple plugin creates Temporary Vector Layer from Delimited Text Layer just in a few seconds that allows you to add, delete, move or edit any feature you want.

<img src="https://github.com/g2r4i6e8/geocsv/blob/main/docs/Problem.png?raw=true" width="800" />

## Usage

1. Add Delimited Text Layer via Layer->Add Layer->Add Delimited Text Layer...
<img src="https://github.com/g2r4i6e8/geocsv/blob/main/docs/Step 1.png?raw=true" width="800" />

2. Open GeoCSV plugin from Toolbar
<img src="https://github.com/g2r4i6e8/geocsv/blob/main/docs/Step 2.png?raw=true" width="800" />

3. Select a Delimited Text Layer you want to transform to Temporary Vector Layer
<img src="https://github.com/g2r4i6e8/geocsv/blob/main/docs/Step 3.png?raw=true" width="800" />

4. Click "OK"

5. Enjoy!
<img src="https://github.com/g2r4i6e8/geocsv/blob/main/docs/Step 4.png?raw=true" width="800" />
